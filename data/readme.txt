# Data

Brief description of what this folder contains and what it's used for.

## Contents

- `cbs_pc4_2024_v1.gpkg` — PC4 level geospatial mapping of each PC4 in the Netherlands (opendata.cbs.nl)
- `indexed_data.csv` — indexed data based on the 7 different indices in sbi_sector_classification_codebook.csv as .csv
- `indexed_data.gpkg` — indexed data based on the 7 different indices in sbi_sector_classification_codebook.csv as .gpkg (kept geometry)
- `indexed_data_polygons.csv`  —
- `indexed_data_polygons.gpkg`  —
- `pc4_2024_v1.xlsx`  —
- `pc4_polygons.json`  —
- `pc4geoloc.json`  —
- `sbi_sector_classification_codebook.csv`  —

## Source

`indexed_data` — created from FIRMbackbone (KVK) data available through Utrecht University
`cbs_pc4_2024_v1`  — obtained from [CBS](opendata.cbs.nl)
